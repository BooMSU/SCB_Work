1. MS Team ขึ้นให้ Update และไม่สามารถเปิดใช้งานได้
=> MS Team อาจจะเป็น version เก่า ให้ถอนการติดตั้งตัวเก่า แล้วทำการ Restart เครื่องคอมพิวเตอร์ หลังจากนั้นติดตั้ง MS Team ตัวใหม่ล่าสุด

2. ต่อหูฟัง Offserve แล้วไม่ได้ยินเสียง
=> ตรวจเช็คเบื้องต้นมาต่อสาย Offserve ถูกหรือไม่ หากถูกต้องแล้วตรวจเช็คว่าหูฟังมีปัญหามั้ย เช็คโดยการเข้า Tab ค้นหา "Sound Setting" แล้วเช็คว่าเสียงหูฟังปกติมั้ย ถ้าเช็คแล้วไม่เจอปัญหาให้ run command "Copy_Chrome.bat" แล้วลองเข้าใช้งานดูใหม่

3. ทำไมคำสั่ง "Copy_Chrome.bat" ตอนกำลัง run อยู่แล้วขึ้นตัวหนังสือสีแดง
=> สาเหตุหลักมาจากพื้นที่ D:\\ เต็ม สามารถเข้าไปเช็คได้ตรง This PC หากพื้นที่ D:\\ ให้แจ้งฝ่าย IT

4. เปิด Salesforce ไม่ได้ หน้าจอขึ้นเป็นสีขาว
=> เช็ค Internet ว่าใช้งานได้มั้ย ถ้าใช้งานได้ให้กด Tab ค้นหา "cmd" -> พิมพ์คำว่า "gpupdate /force" แล้วกด Enter หลังจากนั้น Restart เครื่องคอมพิวเตอร์ แล้วลองเข้าใช้งานดูใหม่อีกครั้ง ถ้ายังใช้งานไม่ได้ให้แจ้งฝ่าย IT

5. ลูกค้าไม่ได้ยินเสียง Agent / Agent ไม่ได้ยินเสียงลูกค้า
=> อาจจะเกิดจากหูฟังมีปัญหา ให้เช็คตัวหูฟังว่าใช้งานได้มั้ย ถ้าใช้งานไม่ได้อาจจจเกิด Driver มีปัญหา แต่หากใช้งานได้ให้วนกลับไปทำข้อที่2

6. Login เข้าเครื่องคอมพิวเตอร์ไม่ได้
=> ตรวจเช็คว่าเครื่องคอมพิวเตอร์ได้ต่อ LAN มั้ย หากต่อแล้วไม่ได้มีปัญหาเรื่อง Internet อาจจะมาจากเครื่องคอมพิวเตอร์หลุด Domain ให้นำเครื่องไป Join Domain ใหม่

7. Connect WIFI ไม่ได้
=> ให้คลิกขวาที่ Wifi ที่จะทำการเชื่อมต่อและ Forgot Wifi ตั้วนั้นและเชื่อมต่อใหม่อีกครั้ง

8. ไม่สามารถ Print เอกสารได้
=> เช็ค printer ว่ามีการ set printer แล้วหรือยัง หากยังไม่ได้ set printer แจ้งฝ่าย IT ให้ดำเนินการ set printer เมื่อเสร็จแล้วให้ดูสถานะการใช้งานของเครื่อง printer ว่ามีสถานะพร้อมใช้งานหรือป่าว

9. ไม่สามารถเข้าใช้งาน BBI และ Nice System ได้
=> แก้ไขด้วยการเปิดหน้ากล้วยหรือพิมพ์ "http://angie.scb.co.th/easy.asp" ที่ Browser -> กดจุด3จุดมุมขวาบน -> Settings -> Default browser -> Internet Explorer compatibility -> IE Mode (เปลี่ยนจาก Default เป็น Allow) -> Internet Explorer mode pages (กด Add) -> จากนั้นกด "http://angie.scb.co.th/easy.asp" มาวาง -> จากนั้นกด Restart ตัว browser แล้วลองเข้าใช้งานใหม่ดูอีกครั้ง

10. ต่อสาย LAN แต่เข้าใช้งาน Internet ไม่ได้
=> อาจจะเกิดจากสาย LAN มีปัญหา หรือเครื่องคอมพิวเตอร์ไม่ได้ Set IP Address -> ดำเนินการ Set IP Address

11. หน้าจอกระพริบ
     -> อาจจะเกิดจากหน้าจอมีปัญหา
     -> เกิดจาก Driver มีปัญหาหรือทำงานผิดพลาด หรือ Driver ไม่ตรงกับตัวรุ่น ให้ลอง Update Driver ใหม่

12. คอมพิวเตอร์เปิดไม่ติด
     -> อาจจะเกิดจากไม่ได้เสียบสายอุปกรณ์
     -> เกิดจาก Hardware มีปัญหา หรือบางตัวเกิดความเสียหาย (หากแก้ไขแล้วยังไม่หายให้แจ้งซ่อม)

13. คอมพิวเตอร์ทำงานช้าลง
     -> ปิดโปรแกรมที่ไม่ได้ใช้งาน เพื่อลดการใช้งานของหน่วยความจำ หากยังช้าอยู่ให้ Update ระบบ
     -> Update Hardware ให้มีประสิทธิภาพสูงขึ้น

14. หน้าจอค้างหรือโปรแกรมไม่ตอบสนอง
=> ตรวจเช็คว่าโปรแกรมไหนที่ค้างและไม่ตอบสนอง เกิดขึ้นบ่อยมั้ย เบื้องต้นแก้ไขโดย
     -> ปิดโปรแกรมที่ค้างโดยใช้ Task Manager (Ctrl+Shift+Esc) แล้วเลือกโปรแกรมที่ต้องการปิด -> กด End Task
     -> Restart เพื่อ Reset ระบบการทำงานของคอมพิวเตอร์ หากปัญหายังอยู่ให้ลอง Update โปรแกรม หรือถอนการติดตั้ง และทำการติดตั้งใหม่

15. จอภาพแสดงผลผิดปกติ สีเพี้ยน จอมืด
=> ตรวจสอบก่อนว่าสายอุปกรณ์ได้มีการเชื่อมต่อหรือหลวมมั้ย ลองเปลี่ยนสายเพื่อทดสอบ หรือลองเปลี่ยน port ไปเชื่อมจออื่นเพื่อดูว่าปัญหาอยู่ที่จอหรือไม่ ตรวจสอบความละเอียดหน้าจอให้ตรงกับรุ่นหน้าจอ และเช็ค Driver ของจอภาพ หากยังมีปัญหาอยู่ อาจจะเป็น Hardware ที่ต้องตรวจสอบการ์ดจอหรือจอภาพ

16. คอมพิวเตอร์ Restart ตัวเองอัติโนมัติ
=> ตรวจสอบว่าได้มีการติดตั้งโปรแกรมใหม่ หรือ Update อะไรบ้างหรือไม่
     -> เกิดจากอุปกรณ์มีความร้อนสูงเกินไป (เพราะถ้าอุปกรณ์มีความร้อนสูงเกินไป มีส่วนทำให้คอมพิวเตอร์ restart ตัวเองอัตโนมัติ)
     -> ตรวจสอบการตั้งค่าของอุปกรณ์ Power Options ว่ามีการตั้งค่าให้ปิดหรือเข้าสู่โหมดประหยัดพลังงานหรือไม่ หากปัญหายังอยู่ อาจจะเกิดจาก Hardware เช่นแหล่งจ่ายไฟ หรือ Mainboard (แจ้ง IT หรือแจ้งซ่อม)

17. หน้าจอเป็นสีฟ้า
     -> สังเกตุ Code ข้อผิดพลาดที่แสดงบนหน้าจอสีฟ้า เพื่อหาวิธีแก้ไขจาก Code
     -> ลอง Restart เครื่องดูว่ายังเกิดปัญหาอยู่หรือไม่
     -> ตรวจสอบเรื่องการ Update Driver หรือโปรแกรมที่อาจจะขัดแย้งกับระบบ ลอง Update หรือลบโปรแกรมที่พึ่งติดตั้งไปก่อนหน้านี้
     -> หากปัญหายังคงอยู่ อาจจะต้องตรวจสอบในเรื่องของ Hardware เพื่อหาข้อผิดพลาด

18. เปิดเครื่องคอมพิวเตอร์แล้วหน้าจอขึ้นเป็นสีฟ้าให้ใส่รหัส Bitlocker
=> สาเหตุมาจากคอมพิวเตอร์ติด Bitlocker ให้แจ้ง IT เพื่อทำการขอ Key เพื่อปลดล็อค Bitlocker เพื่อให้สามารถเข้าใช้งานได้

19. หลังจากใช้งานคอมพิวเตอร์ไปสักพักแล้วได้ยินเสียงพัดลมและเครื่องร้อนขึ้น
=> สาเหตุหลักเกิดจาก
     -> ฝุ่นสะสมภายในอุปกรณ์คอมพิวเตอร์ ทำให้พัดลมต้องทำงานหนักขึ้นเพื่อระบายความร้อน ส่งผลให้เครื่องดังขึ้น -> แก้โดยทำความสะอาดอุปกรณ์เพื่อขจัดฝุ่นในช่องระบายอากาศ
     -> การใช้งานที่หนักเกินไป หรือใช้งานหนักๆ ใช้งานหลายโปรแกรมพร้อมๆกัน ทำให้เครื่องคอมพิวเตอร์ต้องทำงงานหนักจนเกินไป ทำให้เกิดความร้อนสูง -> ต้องตรวจสอบโปรแกรมที่ใช้งานอยู่ และปิดโปรแกรมที่ไม่จำเป็นออก
     -> ซิลิโคนระบายความร้อนเสื่อมสภาพ ส่งผลให้เกิดการระบายความร้อนได้ไม่ดี จำเป็นต้องล้างซิลิโคนเก่าและหาซิลิโคนใหม่ที่มีคุณภาพสูงเพื่อช่วยให้การระบายความร้อนดีขึ้น
     -> หากปัญหายังอยู่ให้ดูเรื่องของ Hardware ว่าคุณภาพดีหรือไม่ หากไม่ดีพอ จำเป็นต้อง Update Hardware เช่น พัดลมระบายความร้อน ฮีตซิงค์ ให้มีประสิทธิภาพมากยิ่งขึ้น
