ปัญหาต่างๆที่เจอเกี่ยวกับการทำงาน
1. ปัญหาเกี่ยวกับการใช้งาน PC Statement View (ภาษาต่างดาว) หรือโปรแกรมมีปัญหา
=> เข้าเว็บกล้วย หรือ angie.scb.co.th/easy.asp (Google Chrome) แล้วหา keyword คู่มือการใช้งาน -> เลือกข้อ4 Update Template -> คลิกขวาแล้วเลือก save link as... -> save เข้าไปที่ Desktop และเปลี่ยนนามสกุลไฟล์เป็น .bat -> แล้วกด save จากนั้นปิดโปรแกรม PC Statement แล้วกดเข้าใหม่ ถ้าไม่หายให้ติดต่อแผนก IT

2. ปัญหาเกี่ยวกับการใช้งานโทรศัพท์ เช่น รับสายไม่ได้ Browser มีปัญหาและอื่นๆ
=> เข้า path \\10.26.4.203 แล้วเลือก IT_Shared_Folder เลือก NTT_Komgrit (ให้กดเข้า browser แล้วให้ update versionให้เป็นล่าสุดก่อน) -> จากนั้นเลือก #Up_Chrome_Beta -> รอให้การทำงานเสร็จ (กดเข้า browser ผู้ใช้งานจะขึ้นเป็นรูปหมูเด้งและ Chrome จะเป็น version ล่าสุด)

3. ปัญหาเกี่ยวการใช้งาน printer เช่น print ไม่ได้
=> อาจจะมีปัญหาเรื่องการ Set Printer ถ้ามีการ Set Printer แล้วให้ลองดูว่าเครื่อง Printer มีสถานะพร้อมใช้งานมั้ย

3.1 การเข้าใช้งาน printer ต่างๆและเพิ่ม User เพื่อเข้าใช้งานเครื่อง Printer
=> 3.1.1 การเพิ่มและตั้งค่า Printer
       => กด Tab ค้นหาและพิมพ์ Print Management -> All Printers -> เลือก printer ที่ต้องการ (ถ้าไม่มีให้ลง driver printer) -> เลือก properties -> การตั้งค่าอุปกรณ์ -> ผู้ดูแลระบบ -> บัญชีงาน -> ใส่หมายเลขผู้ใช้งานจากนั้นกด OK
          -> เลือก Ports -> ต้องตั้งค่า port = IP Printer ของโซนนั้นๆ
          *** เลือก Printer แล้วถ้ากด properties แล้วไม่เข้าหน้าตั้งค่าจะมีหน้าต่างขึ้นมาให้กด No แล้วให้เลือก Driver ของเครื่อง printer ใหม่ (เลือก Advanced -> Driver -> กด New Driver -> เลือก Kyocera หารุ่นใกลเคียงกัน เช่น Kyocera TASKalfa 6002i -> Next -> และกด Finish)
=> 3.1.2 การเพิ่ม User เพื่อเข้าใช้งานเครื่อง Printer
       => (Printer Kyocera) เปิด Browser และพิมพ์ IP Printer ของโซนนั้นๆ -> เข้ารหัส admin -> เลือก Document Box -> เลือก Custom Box -> กด Add -> จำเลขกล่อง -> ใส่รหัสพนักงาน (รวมทั้ง Password และ Confirm Password) -> เลือก on ทั้งหมดและกด Submit
             -> กดเลือก Management Accountings -> Job Accounting -> Add Account -> ใส่ Account Name และ Account ID จากนั้นกด Submit
             -> 3.1.2.1 การตั้งค่าให้ User ถ่ายเอกสารได้
	-> Print Management -> เลือกเครื่องปริ้น Kyocera 5002i -> คลิกขวาเลือก properties -> General -> Preferences -> เลือก "งาน" -> ติ๊กถูกที่ "การเก็บข้อมูลงาน" -> เลือกชนิด "กล่องแบบกำหนดเอง" -> (จะมีปุ่มการตั้งค่าขึ้นมา) คลิกการตั้งค่า -> เลือกใช้หมายเลขกล่อง -> ใส่หมายเลขกล่องและรหัสผ่าน
	*** ถ้าจำหมายเลขกล่องไม่ได้ให้เข้าไปที่ 10.26.2.(...) เลือกของโซนนั้น -> ใส่รหัส Admin -> Document Box -> Custom Box -> ใส่หมายเลขพนักงานใน Box Name (จะมีหมายเลขกล่องอยู่ในนั้น)
       => (Printer Samsung) เปิด Browser และพิมพ์ IP Printer ของโซนนั้น -> เข้ารหัส admin -> เลือก Security -> User Access Control -> Accounting -> Standard Accounting List -> กด Add -> ใส่ Account ID, Password, และ Confirm Password (รหัสพนักงาน) -> เลือก Features สี(แล้วแต่ตำแหน่งว่าจะให้ปริ้นสีหรือขาวดำ) -> กด Apply

4. ปัญหาเกี่ยวกับการใช้งาน FIS และ FBS เป็นภาษาต่างดาว
=> 4.1 การลบ User แล้วสร้างใหม่ (ลองเข้า User อื่นดูก่อนว่าใช้ได้มั้ย)
          -> 4.1.1 เข้า D:// แล้วเลือก User ที่มีปัญหา จากนั้นกดเปลี่ยนชื่อ
          -> 4.1.2 กด Tab ค้นหาพิมพ์ registry Editor -> HKEY_LOCAL_MACHINE -> SOFTWARE -> Microsoft -> Windows NT -> CurrentVersion -> ProfileList -> หา User ที่ต้องการแล้วกดลบ -> หลังจากนั้นให้ Restart เครื่องหนึ่งครั้งเพื่อสร้าง User เดิม(ระบบจะทำการสร้าง User ใหม่ แต่ใช้ชื่อ User เดิม)
=> 4.2 การ Repair Program ใหม่ 
=> 4.3 ถ้าทำทุกอย่างแล้วไม่หายให้ติดต่อ IT

5. ปัญหาเกี่ยวกับการใช้งาน Offserve (อุปกรณ์เสริมหูฟัง)
=> ตรวจเช็คว่าติดตั้งสายถูกหรือไม่ หูฟังหลักจะต้องอยู่ด้านล่างของเครื่อง หูฟังของคนที่ฟังด้วยจะต้องอยู่ด้านขวาของเครื่อง -> ลูกค้าต้องได้ยินเฉพาะเสียงของผู้พูดหูฟังหลัก (ถ้าได้ยินให้ตั้งค่าที่เครื่อง Offserve ให้ลองหมุนดูจะอยู่ในระดับ 1-4) -> หลังจากนั้นให้ปิดโปรแกรม แล้วเปิดใหม่อีกครั้ง

6. ปัญหาเกี่ยวกับการใช้งาน MS Teams
=> 6.1 User ส่งข้อความไม่ได้ (ถ้าไม่ได้มีปัญหาเรื่อง Internet)
      -> ตรวจสอบว่าเครื่องคอมได้เชื่อมต่อ LAN หรือยัง
      -> ให้ลอง Signout และ Login ใหม่อีกครั้ง
=> 6.2 ขึ้นให้ Update หรือเปิดใช้งานไม่ได้
      -> MS Team อาจจะเป็น version เก่า ต้องทำการลบ และติดตั้งใหม่ และ Restart เครื่องใหม่อีกครั้ง

7. ปัญหาเครื่องค้าง หรือ Taskbar ค้าง
=> เข้า cmd -> ใช้คำสั่ง taskkill /f /im explorer.exe && start explorer.exe
