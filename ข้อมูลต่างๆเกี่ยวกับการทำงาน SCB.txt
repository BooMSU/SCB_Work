5. Command ที่ใช้การทำงาน (เปิด cmd ก่อนทุกครั้ง)
     5.1 ipconfig /all -> ใช้ตรวจสอบ IP ของเครื่องคอมพิวเตอร์
     5.2 gpupdate /force -> ใช้เวลาเว็บไซต์มีปัญหา เข้าไม่ได้ แต่คอมพิวเตอร์สามารถเชื่อม Internet ได้
     5.3 taskkill /f /im explorer.exe && start explorer.exe -> ใช้เวลา Taskbar Window ค้าง
